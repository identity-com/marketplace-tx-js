(inner) getTransactionCount(address, defaultBlock) → {Promise.<number>}

Source:
  • support/nonce.js, line 109

Returns current transaction count for given address.

NOTE: There are reports of incorrect behaviour of web3.eth.getTransactionCount which affects the count of pending transactions. https://github.com/ethereum/go-ethereum/issues/2880 At this time we could only rely on the count of mined transactions.

Parameters:
Name Type Description
address

The address to get the numbers of transactions from.
defaultBlock

The default block number to use when querying a state. "earliest", the genesis block "latest", the latest block (current head of the blockchain) "pending", the currently mined block (including pending transactions)
